# Changelog

All notable changes to the Protagons open standard will be documented in this file.

## [1.0.0] - 2026-03-04

### Added
- Initial public release of the Protagons open standard
- JSON Schema v1.0.0 (`schema/character.schema.json`) — 28 dimensions, 287 fields
- Example profile: Burned Out Ops (`profiles/professional/`)
- Validator tool (`tools/validator.py`)
- Compiler tool (`tools/compiler.py`)
- OpenClaw integration: SOUL.md compiler, batch compiler, example output
- Documentation: SPEC, dimensions, rationale, API, deployment, integrations
- Dual licensing: MIT (schema/tools), CC-BY-4.0 (profiles)
