# Chat Platform Examples

Use Protagon identities with chat platforms.

## Examples

| File | Platform | Description |
|------|----------|-------------|
| `custom_gpt_instructions.md` | OpenAI Custom GPTs | Paste as GPT instructions |
| `poe_bot_setup.md` | Poe | Server bot configuration |

## How it works

Most chat platforms accept a "system prompt" or "instructions" field.
Copy the `synthesized_prompt.content` from a `.protagon.json` file and
paste it into the platform's configuration.
