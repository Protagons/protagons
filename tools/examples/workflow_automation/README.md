# Workflow Automation Examples

Integrate Protagons into workflow automation platforms.

## Examples

| File | Platform | Description |
|------|----------|-------------|
| `n8n_workflow.json` | n8n | Self-hosted workflow with HTTP nodes |
| `make_scenario.json` | Make.com | HTTP module calling Protagons API |

## How it works

These platforms call the Protagons API via HTTP request nodes.
You can fetch library items, trigger generations, or deploy to platforms.

## API Base URL

```
https://api.usaw.ai/api/v1
```

See the [API reference](https://protagons.com/generate) for full endpoint documentation.
